A KVM (Kernel-based Virtual Machine) is a virtual private server which sits on top of a physical dedicated server. Resources are dedicated to the KVM VPS and are not shared with other users on the physical device, offering tremendous reliability and performance. A KVM is true virtualization where the VPS operates as its own server, independently of the host node.
